# banben

A better `npm version`. The word "banben" comes from the Chinese word `版本` (bǎnběn), meaning "version".

## Installation

```sh
npm -g i banben
pnpm -g add banben
yarn -g add banben
bun -g add banben
```

## Usage

```sh
banben [<version> | major | minor | patch | premajor | preminor | prepatch | prerelease]
```
